# Cadeaus
Natuurlijk vinden we het leuk om een huwelijkscadeau te krijgen! Als je op zoek bent naar inspiratie ben je hier op de goede plek.

Via deze [link](https://www.lijstje.nl/huwelijk-van-aa) kom je op een pagina met cadeaus waar we blij van worden.

We krijgen graag (onderdelen van) een huwelijksservies. Op het lijstje hebben we linkjes gezet naar de officiële webshop van Bitz. Kijk ook even verder rond, want we zien hetzelfde servies op andere sites, soms goedkoper of met korting. We willen alleen servies van de set waarbij de buitenkant mat-zwart is.  

Let op dat er tijdens onze huwelijksreis niet bij ons thuis bezorgd kan worden. We zijn op reis van 20 mei tot 26 juni en **kunnen dan geen pakketjes aannemen**.

## Huwelijksreis
Je kan ons ook een onderdeel van onze huwelijksreis naar Nieuw Zeeland cadeau doen. We gaan er prachtige avonturen beleven waar je ons een bijdrage aan kan geven via de linkjes op het lijstje.  
Updates van onze avonturen kan je volgen via deze [Polarsteps](https://www.polarsteps.com/HuwelijksreisAA/24257938-huwelijksreis-new-zealand?s=9936ea7b-60e2-4aef-b91c-79580bf0eae4).
