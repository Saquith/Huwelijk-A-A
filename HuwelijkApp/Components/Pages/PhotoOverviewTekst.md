# Foto’s
Wat is een huwelijksfeest zonder throwback-slideshow?!  
Via onderstaande link kan je foto’s van jouw herinneringen met ons uploaden. We zullen deze tijdens het feest op een slideshow laten zien.

Als je tijdens het feest een aantal van jouw leukste foto’s uploadt, zullen die op de slideshow terechtkomen.

Via hetzelfde album kan je ook na afloop je gemaakte foto’s uploaden. We vinden het leuk om deze te ontvangen!
