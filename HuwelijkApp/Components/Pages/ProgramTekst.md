# Programma
Op de achterkant van jouw kaart staat voor welk gedeelte je bent uitgenodigd.
Het volledige programma luidt als volgt:

| _Aankomst met je [carpool-buddies](/info#waar-kan-ik-parkeren)_ |       |
|-----------------------------------------------------------------|-------|
| Welkom en lopende lunch                                         | 11:30 |
| Ceremonie                                                       | 12:30 |
| _Toost en taart_                                                |       |
| _Foto’s en vertier_                                             |       |
| Avondeten                                                       | 18:00 |
| Aanvang feest                                                   | 20:00 |
| Openingsdans                                                    | 20:30 |
| Wickerwens                                                      | 22:00 |

Na de Wickerwens gaat het feest door met wie dat wil, met gezelligheid, muziek, vuur en dans tot wanneer je wil. Je kan [blijven slapen](/overnachten).
